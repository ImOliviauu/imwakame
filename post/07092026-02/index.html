博客主题自定义

2026-07-09

博主在 simple 主题给定的自定义范畴之外的一些修改。

完整头图

博主写影评通常会将海报（通常是竖版的）作为文章头图（也就是封面），但是 simple 会默认把头图截成 16:9 的。如果想要将头图能完整且自适应大小（不留白边），可以做以下修改。

1. 修改 post.ejs

\simple\templates\post.ejs
找到：

<% if (post.feature) { %>
  <div class="feature-container" style="background-image: url('<%= post.feature %>')">
  </div>
<% } %>

替换成：

<% if (post.feature) { %>
  <div class="feature-container">
    <img src="<%= post.feature %>">
  </div>
<% } %>

2. 修改 main.css

\simple\assets\styles\main.css
找到：

.post-detail .feature-container {
  padding-top: 56.25%;
  background-size: cover;
  background-position: center;
  border-radius: 3px;
  box-shadow: 0 2px 5px rgba(0, 0, 25, 0.1), 0 5px 75px 1px rgba(0, 0, 50, 0.2);
  margin-bottom: 24px;
}

替换成：

.post-detail .feature-container {
  width: 100%;
  border-radius: 3px;
  overflow: hidden;
  margin-bottom: 24px;
  box-shadow: 0 2px 5px rgba(0, 0, 25, 0.1), 0 5px 75px 1px rgba(0, 0, 50, 0.2);
}
.post-detail .feature-container img {
  display: block;
  width: 100%;
  height: auto;
}

添加评论

pro 支持了 cusdis ，搞起来简单一点
这个跟着官方文档走就可以，如果你的主题没有显示评论框，就复制一下官网给的那一段 html 家在 post.ejs 正文部分后面，如果发现评论框很矮，在 style 的 main.css 里加入：

#cusdis_thread {
  width: 100%;
  min-height: 400px;
}
#cusdis_thread iframe {
  width: 100% !important;
  min-height: 400px !important;
  border: none;
}
